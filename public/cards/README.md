# Cards

Esta pasta é destinada aos assets de cartas utilizados pelo jogo.

Por questões de licenciamento e direitos autorais, os arquivos de cartas não são distribuídos junto com este repositório.

Adicione aqui os arquivos das cartas que deseja utilizar.

## Recomendações

* Utilize nomes sem espaços.
* Evite caracteres especiais.
* Mantenha a nomenclatura compatível com o Card Manager do projeto.
* Você pode criar seu próprio pacote de cartas seguindo a convenção de nomes abaixo.
